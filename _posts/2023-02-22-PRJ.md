### 터미널에 명령어를 입력해서 설치.

{% highlight java %}

1. yarn으로 설치

yarn add react-bootstrap bootstrap

2. npm으로 설치
 
npm install react-bootstrap bootstrap

[주의사항]

yarn 또는 npm으로 설치시 index.js에

import 'bootstrap/dist/css/bootstrap.css';

추가한다.

{% endhighlight %}

### 적용법

bootstrapk.com에서 원하는 UI를 가져와서 사용하면 된다.
